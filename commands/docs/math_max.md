# <code>{{ $frontmatter.title }}</code> for math

<div class='command-title'>{{ $frontmatter.math }}</div>

## Signature

```> math max ```

## Examples

Find the maximum of list of numbers
```shell
> [-50 100 25] | math max
100
```

Find the maxima of the columns of a table
```shell
> [{a: 1 b: 3} {a: 2 b: -1}] | math max
╭───┬───╮
│ a │ 2 │
│ b │ 3 │
╰───┴───╯
```
