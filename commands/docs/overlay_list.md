# <code>{{ $frontmatter.title }}</code> for core

<div class='command-title'>{{ $frontmatter.core }}</div>

## Signature

```> overlay list ```

## Notes
The overlays are listed in the order they were activated.
## Examples

Get the last activated overlay
```shell
> module spam { export def foo [] { "foo" } }
    overlay use spam
    overlay list | last
spam
```
