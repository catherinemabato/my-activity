# <code>{{ $frontmatter.title }}</code> for math

<div class='command-title'>{{ $frontmatter.math }}</div>

## Signature

```> math arccos --degrees```

## Parameters

 -  `--degrees` `(-d)`: Return degrees instead of radians

## Examples

Get the arccosine of 1
```shell
> 1 | math arccos
0
```

Get the arccosine of -1 in degrees
```shell
> -1 | math arccos -d
180
```
