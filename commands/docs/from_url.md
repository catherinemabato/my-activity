## Signature

```> from url ```

## Examples

Convert url encoded string into a record
```shell
> 'bread=baguette&cheese=comt%C3%A9&meat=ham&fat=butter' | from url
╭────────┬──────────╮
│ bread  │ baguette │
│ cheese │ comté    │
│ meat   │ ham      │
│ fat    │ butter   │
╰────────┴──────────╯
```
