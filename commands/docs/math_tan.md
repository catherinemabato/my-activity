# <code>{{ $frontmatter.title }}</code> for math

<div class='command-title'>{{ $frontmatter.math }}</div>

## Signature

```> math tan --degrees```

## Parameters

 -  `--degrees` `(-d)`: Use degrees instead of radians

## Examples

Apply the tangent to π/4
```shell
> (math pi) / 4 | math tan
1
```

Apply the tangent to a list of angles in degrees
```shell
> [-45 0 45] | math tan -d
╭───┬─────────╮
│ 0 │ -1.0000 │
│ 1 │  0.0000 │
│ 2 │  1.0000 │
╰───┴─────────╯

```
