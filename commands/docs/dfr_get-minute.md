## Signature

```> dfr get-minute ```

## Examples

Returns minute from a date
```shell
> let dt = ('2020-08-04T16:39:18+00:00' | into datetime -z 'UTC');
    let df = ([$dt $dt] | dfr into-df);
    $df | dfr get-minute
╭───┬────╮
│ # │ 0  │
├───┼────┤
│ 0 │ 39 │
│ 1 │ 39 │
╰───┴────╯

```
