## Examples

Add an index to each element of a list
```shell
> [a, b, c] | enumerate
╭───┬──────╮
│ # │ item │
├───┼──────┤
│ 0 │ a    │
│ 1 │ b    │
│ 2 │ c    │
╰───┴──────╯

```
