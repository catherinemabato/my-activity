upcases text

## Usage

```shell
> str upcase ...args {flags}
```

## Parameters

- ...args: optionally upcase text by column paths

## Flags

- -h, --help: Display this help message

## Examples

Upcase contents

```shell
> echo 'nu' | str upcase
```
