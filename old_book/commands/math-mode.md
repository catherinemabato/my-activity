Gets the most frequent element(s) from a list of numbers or tables

## Usage

```shell
> math mode {flags}
```

## Flags

- -h, --help: Display this help message

## Examples

Get the mode(s) of a list of numbers

```shell
> echo [3 3 9 12 12 15] | math mode
```
